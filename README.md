# Apex Region Plugin
Chart Line for Oracle Apex.


# How to Use
1. Create a Region
2. Choose Type "Chartjs Line [Plug-In]"
3. Sql Query Exampe
```
select EMPNO data, 
ename label  
from emp_ 

```

4. Under Attributes: - you can select :

- the Chart Title

- background Color

 - Border Color                                        
                                        
 - the Chart Label
